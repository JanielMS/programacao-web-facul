Informações Complementares

EMENTA
Conhecer Linguagens de Programação para desencolvimento de aplicações web; construir web sites dinâmicos baseados em novas tecnoligias e com acesso a banco de dados; compreender os métodos e técnicas de desenvolvimento de aplicações avançadas para web; identificar soluções na otimização e melhoria do desempenho de web sites; compreender o cundionamento básico de um servidor de internet.
OBJETIVOS
GERAL: Propiciar uma formação consistente promovendo a prática de conceitos de programação WEB; uso de banco de dados relacional e eleboraçção ede projeto de um site web dinâmico. ESPECÍFICOS: Conhecer Linguagens de programação e marcação, dos lados cliente e servidor; Elaborar modelo conceitual de banco de dados; Criar um site dinâmico usnado linguagem de programação web e banco de dados;
